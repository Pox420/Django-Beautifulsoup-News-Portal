Front-end

  • HTML
  • Bootstrap

Database

  • Sqlite3

Backend

  • Python 3
  • BeautifulSoup
  • Django Rest Framework

Let's see the functionality in this Project .

  • Data: Data is scraped from various website's using BeautifulSoup and Scraped data is stored in Database.
  • User: Modified the user model using AbstractUser to add a email and password.
  • User Registration: You can register using Email and password.
  • User Login: You can login using Email and Password.
  • Verify Email: You need to Verify your Email to login.
  • Forgot Password: You can reset your Password if needed.
  • Form: Form is created by using UserCreationForm which is Inherited from django.contrib.auth.forms
  • Mail: Mail is sent using EmailBackend which is Inherited from django.core.mail
  • Logout: You can logout from the Website using Logout button.
